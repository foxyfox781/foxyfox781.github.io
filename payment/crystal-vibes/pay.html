Information importante avant paiement

Afin de pouvoir attribuer correctement l’accès, le rôle ou l’avantage associé à votre paiement, il est obligatoire d’indiquer votre Discord lors du paiement.

⚠️ Lors du paiement, merci d’indiquer votre Discord ID ou votre nom d’utilisateur Discord dans le champ « Nom ».

Si le Discord renseigné est introuvable, invalide ou incorrect, l’accès ne pourra pas être attribué immédiatement.

Dans ce cas, un remboursement pourra être effectué ultérieurement, après vérification par l’équipe Univerbeta.

Foxy • Foxy#1234 • ID Discord
